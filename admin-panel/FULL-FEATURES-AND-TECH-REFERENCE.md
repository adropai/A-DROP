# 📦 مستند جامع قابلیت‌ها و تکنولوژی‌های پروژه A-DROP (Admin Panel)

## 🏢 معرفی کلی
پروژه A-DROP یک سیستم مدیریت رستوران جامع است که شامل:
- **داشبورد ادمین (Admin Panel)**
- **داشبورد سوپر ادمین (Super Admin Panel)**
- **اپلیکیشن مشتری (Customer App)**
- **وب‌سایت فروش (Marketing Website)**

این مستند، تمام قابلیت‌ها، تکنولوژی‌ها و استانداردهای طراحی را به صورت کامل شرح می‌دهد تا توسعه سایر بخش‌ها (سوپر ادمین، مشتری، وب‌سایت) با همین مسیر و کیفیت انجام شود.

---

## 🧩 ساختار و قابلیت‌های کامل داشبورد ادمین

### 1. 🏠 داشبورد اصلی
- آمار کلیدی (درآمد، سفارشات، مشتریان)
- نمودارهای تحلیلی (فروش، روند هفتگی)
- جدول سفارشات فعال
- پنل اعلانات و اقدامات سریع

### 2. 🔐 احراز هویت و امنیت
- ورود/ثبت‌نام با رمزنگاری
- مدیریت نشست‌ها و دو مرحله‌ای
- بازیابی رمز عبور
- کنترل دسترسی بر اساس نقش و سطح دسترسی

### 3. 👥 مدیریت نقش‌ها و دسترسی‌ها
- 9 نقش پیش‌فرض (SUPER_ADMIN, ADMIN, MANAGER, ...)
- تخصیص دسترسی‌های دانه‌ای به هر کاربر
- مدیریت کاربران بر اساس نقش
- میدل‌ویر امنیتی برای APIها

### 4. 📋 مدیریت سفارشات
- لیست سفارشات با فیلتر و جستجو
- جزئیات سفارش، تغییر وضعیت، چاپ فاکتور

### 5. 🍽️ مدیریت منو
- مدیریت دسته‌بندی و آیتم‌های منو
- آپلود تصویر، قیمت‌گذاری، تخفیف

### 6. 👩‍🍳 پنل آشپزخانه
- نمایش سفارشات در انتظار
- مدیریت صف و تایمر پخت

### 7. 💰 سیستم صندوق
- باز/بسته کردن شیفت
- ثبت تراکنش و گزارش روزانه

### 8. 🪑 مدیریت میزها
- لیست و وضعیت میزها
- QR Code میزها
- مدیریت رزرو

### 9. 📅 مدیریت رزرو
- لیست و تقویم رزرو
- بررسی دسترسی میز و تایید/لغو

### 10. 👥 مدیریت مشتریان
- پروفایل، تاریخچه سفارش، امتیاز وفاداری

### 11. 👨‍💼 مدیریت کارکنان
- لیست، شیفت، نقش و آمار عملکرد

### 12. 📦 مدیریت موجودی
- فهرست کالا، هشدار کمبود، گزارش حرکت

### 13. 🚚 مدیریت تحویل (Delivery)
- مدیریت پیک‌ها، ردیابی تحویلات، آمار

### 14. 📈 آنالیتیکس پیشرفته
- داشبورد تحلیلی، گزارش‌های تفصیلی، پیش‌بینی فروش

### 15. 📢 بازاریابی
- کمپین، کوپن، بخش‌بندی مشتریان، ایمیل مارکتینگ

### 16. 🏢 مدیریت شعبه‌ها (Multi-Branch)
- مدیریت چند شعبه، کارکنان هر شعبه، تنظیمات اختصاصی

### 17. ⚙️ تنظیمات سیستم
- تنظیمات عمومی، برندینگ، پیکربندی

### 18. 🔒 امنیت
- لاگ امنیتی، مدیریت دسترسی، تنظیمات امنیتی

### 19. 🎓 آموزش هوش مصنوعی
- مدیریت مدل‌ها، آموزش، تحلیل داده

### 20. 🔗 یکپارچگی‌ها
- اتصال به سرویس‌های خارجی، API Management، Webhook

### 21. 🎧 پشتیبانی
- تیکت، سوالات متداول، راهنما، چت آنلاین

## 🛠️ تکنولوژی‌ها و کتابخانه‌های استفاده شده

### Frontend
- **Next.js 14.2.15** (React 18)
- **TypeScript**
- **Ant Design** (کامپوننت‌های UI)
- **Tailwind CSS** (استایل پایه و utility)
- **Day.js** (مدیریت تاریخ)
- **SWR/React Query** (دریافت داده)
- **Axios** (در صورت نیاز به درخواست HTTP)
- **react-chartjs-2** و **Recharts** (نمودارها)
- **react-qr-code** (تولید QR)
- **react-pdf** (خروجی PDF)
- **react-toastify** (اعلان)
- **react-dropzone** (آپلود فایل)
- **react-icons** (آیکون)
- **classnames** (مدیریت کلاس‌ها)
- **Jest/React Testing Library** (تست)

### Backend
- **Prisma ORM** (مدیریت دیتابیس)
- **SQLite** (قابل جایگزینی با PostgreSQL/MySQL)
- **bcryptjs** (رمزنگاری)
- **jsonwebtoken** (JWT)
- **nodemailer** (ایمیل)
- **zod/yup** (اعتبارسنجی داده)
- **helmet** (امنیت HTTP)
- **cors** (Cross-Origin)
- **morgan** (لاگ)

### سایر
- **ESLint/Prettier** (کدنویسی تمیز)
- **Husky/Lint-staged** (پیش‌کامیت)
- **dotenv** (مدیریت env)
- **Vercel/Netlify** (استقرار)

## 🎨 استانداردهای طراحی و UI
- **فونت**: Vazirmatn (یا IranYekan)، Roboto برای انگلیسی

- **دکمه‌ها و فرم‌ها**: با Ant Design و Tailwind

## 🟡 درباره 2 درصد باقی‌مانده
1. **Database Migration**: باید migration نهایی برای مدل‌های جدید (شعبه، کارکنان شعبه و ...) اجرا شود:
   ```bash
   npx prisma migrate dev --name "add-branch-system"
   npx prisma generate
   ```
2. **تست نهایی عملکرد**: تست کامل همه APIها و صفحات در محیط staging/production
